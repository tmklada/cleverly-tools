## When Would You Need to Split a PDF?

Here are the most common real-world reasons people split PDFs:

- **Sharing only part of a document** — sending a specific chapter from a larger report
- **Reducing file size** — instead of compressing, just extract the pages you need
- **Separating invoices** — a billing system exports all invoices in one PDF; you split them per client
- **Extracting a cover page** — pulling just the first page for a thumbnail or preview
- **Reordering a document** — split into single pages, then merge in a new order
- **Printing only specific pages** — easier to print when pages are their own files

## Frequently Asked Questions

**Q: Can I split a PDF that has a password on it?**
Password-protected PDFs need to be unlocked first before they can be split. Remove the password protection, then use the splitter.

**Q: Will splitting a PDF reduce the quality?**
No. Splitting only separates pages — it doesn't change the content, resolution, or quality of any page.

**Q: How many pages can the tool handle?**
The tool works well with PDFs of hundreds of pages. The main limitation is file size (up to 100MB per upload).

**Q: What if I accidentally split the wrong pages? Can I undo it?**
The original file is never changed — you just download new files. Simply go back and run the split again with the correct page selection.
